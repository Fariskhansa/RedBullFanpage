# Red Bull Racing F1 Team - Interactive Fanpage

Sebuah halaman web interaktif dan responsif yang didedikasikan untuk tim Formula 1 Red Bull Racing. Proyek ini dibangun untuk menampilkan sejarah, pencapaian, evolusi mobil, dan profil pembalap dengan balutan desain modern yang dinamis. 

Proyek ini sangat menonjolkan interaktivitas pengguna melalui penggunaan animasi CSS tingkat lanjut dan manipulasi DOM menggunakan Vanilla JavaScript murni tanpa bergantung pada framework berat.

## Fitur Utama

* **Desain Responsif & Glassmorphism:** Tampilan web dapat menyesuaikan dengan mulus di layar laptop maupun handphone (dilengkapi dengan Hamburger Menu khusus mobile). Navigasi menggunakan efek kaca buram yang modern.
* **Animasi Scroll (Intersection Observer):** Elemen-elemen web (teks, gambar, kartu) akan muncul secara perlahan dan elegan hanya ketika elemen tersebut mulai terlihat di layar, sehingga menghemat memori browser.
* **Penghitung Angka Otomatis (Number Counter):** Angka statistik pencapaian tim akan menghitung maju secara otomatis saat pengguna menggulir layar ke bagian pencapaian.
* **Partikel Energi Canvas (Background):** Latar belakang halaman utama dilengkapi dengan animasi partikel bergerak yang dirender langsung menggunakan HTML5 Canvas.
* **Efek 3D Tilt Interaktif:** Kartu profil pembalap akan miring secara tiga dimensi mengikuti pergerakan kursor mouse pengguna.
* **Grafik Statistik (Chart.js):** Visualisasi data perjalanan poin konstruktor Red Bull Racing dari tahun ke tahun menggunakan grafik garis interaktif.
* **Popup Modal Profil:** Mengklik kartu pembalap akan meredupkan latar belakang dan memunculkan jendela popup berisi detail profil pembalap di tengah layar.
* **Progress Bar Telemetri:** Garis merah menyala di bagian paling atas layar yang menunjukkan seberapa jauh halaman telah digulir.
* **Efek Audio Pintar:** Suara deru mesin V10 F1 akan berbunyi secara otomatis saat pengguna melihat bagian evolusi mobil untuk pertama kalinya.
* **Efek Mengetik (Typewriter):** Judul utama di halaman depan muncul seolah-olah sedang diketik secara langsung.

## Teknologi yang Digunakan

* **HTML5:** Struktur semantik web.
* **CSS3:** Styling, Flexbox, CSS Grid, animasi Keyframes, dan media queries untuk responsivitas.
* **Vanilla JavaScript:** Logika animasi, penanganan event, dan manipulasi DOM.
* **Chart.js:** Library eksternal untuk render grafik statistik pencapaian.

## Cara Menjalankan Proyek

Proyek ini tidak memerlukan proses instalasi atau build tools khusus. Anda dapat langsung menjalankannya di komputer Anda.

1. Clone repositori ini ke komputer lokal Anda.
2. Buka folder proyek hasil clone.
3. Klik dua kali pada file `index.html` untuk membukanya di browser web pilihan Anda (Google Chrome, Firefox, Safari, dll).
4. Pastikan Anda terhubung ke internet agar gambar dari Wikimedia dan library Chart.js dapat dimuat dengan sempurna.
